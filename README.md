# Figma Land
## Presentation
Figma Land is a project developed to exercise my skills in HTML and CSS, applying BEM architecture.

## Getting Started

```bash
  npm run dev
  # or
  yarn dev
```

## Stacks
- HTML
- CSS
- Vite (to run project locally)
